# مثال‌های اجرایی پیک (Peik Examples)

این پوشه حاوی اسکریپت‌هایی برای تست و نمایش قابلیت‌های هسته مستقل پیک (Peik Core) در محیط Node.js است.

## پیش‌نیازها

- **Node.js**: نسخه 18 یا بالاتر (برای پشتیبانی از `fetch` و `AbortController` داخلی).
- یک کلید API معتبر از گوگل (Gemini).

## اجرای آزمون هسته (`testCore.js`)

این اسکریپت یک چرخه کامل از ایجاد نمونه پیک، ثبت افزونه‌ها، و ارسال یک پیام به Gemini را شبیه‌سازی می‌کند.

### روش اجرا

۱. در ترمینال به پوشه ریشه پروژه بروید.
۲. دستور زیر را اجرا کنید (کلید API خود را جایگزین کنید):

**در لینوکس/مک:**
```bash
API_KEY="کلید_خود_را_اینجا_بگذارید" node examples/testCore.js
```

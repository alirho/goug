# پرسش‌های متداول (FAQ)

---

### ۱. داده‌های من کجا ذخیره می‌شوند؟
تمام داده‌ها (شامل تاریخچه چت‌ها و کلیدهای API) در مرورگر خود شما و با استفاده از **IndexedDB** ذخیره می‌شوند. هیچ داده‌ای به سرورهای پیک فرستاده نمی‌شود، زیرا پیک اصلاً سرور backend ندارد.

### ۲. آیا استفاده از پیک رایگان است؟
پیک به خودی خود یک نرم‌افزار متن‌باز و رایگان است. اما برای استفاده از مدل‌های زبانی (مثل GPT-4 یا Gemini)، ممکن است نیاز به پرداخت هزینه به ارائه‌دهندگان آن سرویس‌ها (Google یا OpenAI) داشته باشید.

### ۳. خطای "Failed to fetch" یا خطای شبکه به چه معناست؟
این خطا معمولاً مربوط به اتصال اینترنت است.
*   اتصال اینترنت خود را بررسی کنید.
*   اگر از ایران متصل می‌شوید، احتمالاً نیاز به ابزارهای عبور از تحریم (DNS یا VPN) دارید تا بتوانید به APIهای گوگل یا OpenAI متصل شوید.

### ۴. آیا می‌توانم از مدل‌های محلی (Offline) استفاده کنم؟
بله! با استفاده از ابزارهایی مثل **Ollama** و قابلیت **Custom Provider** در پیک، می‌توانید مدل‌هایی مثل Llama 3 یا Mistral را روی کامپیوتر خود اجرا کرده و در پیک با آن‌ها چت کنید. کافیست آدرس API را (معمولاً `http://localhost:11434/v1/chat/completions`) در تنظیمات وارد کنید.

### ۵. تفاوت پیک با نسخه اصلی ChatGPT چیست؟
پیک یک رابط کاربری (Client) است که به شما اجازه می‌دهد همزمان از چندین سرویس مختلف استفاده کنید، مدل‌های محلی را اجرا کنید و کنترل کامل روی داده‌های خود داشته باشید. همچنین پیک قابلیت‌های شخصی‌سازی بیشتری دارد.

# شروع کار با پیک (Getting Started)

به دنیای پیک خوش آمدید! پیک یک دستیار هوش مصنوعی شخصی است که مستقیماً در مرورگر شما اجرا می‌شود.

---

## پیش‌نیازها

برای اجرای پیک نیاز به نصب نرم‌افزار پیچیده‌ای ندارید. تنها موارد زیر لازم است:
1.  یک مرورگر مدرن (Chrome, Firefox, Edge, Safari).
2.  اتصال اینترنت.
3.  یک کلید API (از گوگل Gemini یا OpenAI).

---

## نصب و اجرا

چون پیک کاملاً کلاینت-ساید است، نیازی به سرورهای سنگین ندارد.

### روش ۱: استفاده از مخزن گیت (پیشنهادی)
1.  ترمینال را باز کنید.
2.  مخزن را کلون کنید:
    ```bash
    git clone https://github.com/your-username/peik.git
    cd peik
    ```
3.  یک سرور محلی ساده اجرا کنید:
    *   اگر Python دارید: `python -m http.server 3000`
    *   اگر Node.js دارید: `npx serve .`
4.  مرورگر را باز کنید و به آدرس `http://localhost:3000` بروید.

---

## اولین گپ شما (Tutorial)

1.  **تنظیمات اولیه**:
    *   هنگامی که برنامه باز می‌شود، پنجره تنظیمات نمایش داده می‌شود.
    *   کلید API خود (مثلاً Gemini API Key) را در فیلد مربوطه وارد کنید.
    *   روی دکمه **ذخیره تغییرات** کلیک کنید.

2.  **ایجاد گپ**:
    *   روی دکمه **گپ جدید** در نوار کناری کلیک کنید.
    *   یک عنوان پیش‌فرض انتخاب می‌شود، می‌توانید بعداً آن را تغییر دهید.

3.  **ارسال پیام**:
    *   در کادر پایین صفحه بنویسید: "سلام! خودت را معرفی کن."
    *   اینتر بزنید یا دکمه ارسال را فشار دهید.
    *   پیک شروع به تایپ پاسخ خواهد کرد.

---

## نکات عیب‌یابی

*   **پیام خطا در کنسول (CORS)**: مطمئن شوید که برنامه را از طریق `http://` اجرا می‌کنید، نه با باز کردن مستقیم فایل `index.html` (پروتکل `file://` به دلیل محدودیت‌های امنیتی ماژول‌ها کار نمی‌کند).
*   **کلید API کار نمی‌کند**: بررسی کنید که آیا نیاز به VPN دارید (برخی سرویس‌ها مثل گوگل و OpenAI ممکن است IP ایران را محدود کرده باشند).
